# markdown blog by next.js 14

## Architecture

- next.js
- contentlayer
- shadcn/ui

## Features

- [x] markdown blog
- [x] dark mode
- [x] code highlight
- [ ] toc
- [ ] search
- [ ] sitemap

## How to use

```bash
# install dependencies
pnpm i

# run dev server
pnpm dev

# build
pnpm build
```
